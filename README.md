## Features


 - About *Us page
 - Shop page
 - Contact page
 - Blog page
 - Fully Responsive Base Theme (Demo) 
 - Social Icons 
 - Font Awesome
 - Footer navigation
 - Newsletter subscription in Home page
 - Free & Automatic Hosting with GitHub Pages 


## Install Locally

Clone the project

```bash
  git clone https://github.com/hawra-alramadan/Ecommerce-beauty.git
```

Go to the project directory

```bash
  cd Ecommerce-beauty
```
